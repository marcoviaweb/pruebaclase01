Git Master - cambio 01 - main

Tu guía esencial de comandos para el control de versiones

Configuración

Identidad de Usuario

Configura tu nombre y email para tus commits.

git config --global user.name "Tu Nombre"
Iniciar Repositorio

Inicializa un nuevo repositorio Git en el directorio actual.

git init
Clonar Repositorio

Descarga un repositorio existente desde una URL (GitHub, GitLab, etc.).

git clone <url>

Flujo de Trabajo Básico

Ver Estado

Muestra los archivos modificados, agregados o eliminados.

git status
Añadir Archivos

Añade cambios al área de preparación (stage) para el próximo commit.

git add .
Guardar Cambios

Guarda los cambios preparados con un mensaje descriptivo.

git commit -m "Mensaje"

Ramas (Branches)

Listar/Crear Ramas

Muestra todas las ramas locales o crea una nueva.

git branch [nombre]
Cambiar Rama

Cambia a la rama especificada.

git checkout <nombre-rama>
Fusionar Ramas

Combina la historia de una rama en la rama actual.

git merge <nombre-rama>

Sincronización

Enviar Cambios

Envía los cambios de tu rama local al repositorio remoto.

git push origin <rama>
Traer Cambios

Descarga y fusiona los cambios del remoto a tu rama actual.

git pull origin <rama>
Ver Historial

Muestra el historial de commits.

git log --oneline
